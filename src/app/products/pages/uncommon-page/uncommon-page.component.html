Pipes incluidos en Angular por default que nos sirven para tareas muy especificas.

Selector genérico que muestra la cadena que coincide con el valor actual. Necesita un map para verificar coincidencias y retornar el valor de cambio o un valor por defecto en caso de no haber coincidencias.

Estructura

{{i18nSelectForm}}

Ejemplo

Hola buen día, {{name}} es un placer {{gender | i18nSelect:invitationMap}} a nuestro evento.

Selector genérico que mapea un valor y pluraliza el valor resultante de acuerdo a las reglas locales.

Estructura

{{i18nPluralForm}}

Ejemplo

Actualmente {{ clients.length | i18nPlural:clientsMap }}

Crea un nuevo Array o String que contiene un subconjunto (slice) de los elementos declarando una posición inicial y una posición final.

Estructura

{{sliceForm}}

Ejemplo Original

{{clients}}

slice 0:2 
{{clients | slice:0:2  }}

Estructura

{{jsonForm}}

Ejemplo

{{ person | json }}

Suscribe a un Observable o Promise y devuelve el último valor que ha emitido. Cuando se emite un nuevo valor.
Cuando el componente se destruye Async pipe se desuscribe automáticamente para evitar posibles fugas de memoria. Cuando la referencia de la expresión cambia, Async pipe se desuscribe automáticamente del Observable o Promise antiguo y se suscribe al nuevo.

Estructura

{{asyncForm}}

Ejemplo 
      Observlable: {{ myObservalbeTimer | async }}
    
      Promise:
      {{promiseValue | async }}
