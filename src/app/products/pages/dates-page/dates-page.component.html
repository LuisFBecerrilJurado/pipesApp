Pipes incluidos en Angular por default que nos sirven para transformar una cadena de caracteres (string), Número, Booleano o Simbolo.


A un formato que se ajuste a nuestras necesidades, además cada pipe contiene varias opciones para que el resultado obtenido se ajuste a un más a nuestros requerimientos entre los más utilizados se encuentran

{{col.header}} {{param[col.field]}}

    Fecha Actual

    {{customDate}}

  1. Usando date sin formatos

    {{customDate| date }}

  2. Usando date con formato long

    {{ customDate | date:'long' }}

  3. Usando date con formato fullDate

    {{customDate | date:'fullDate'}}

    4. Podemos Agregar otros tipos de configuracion para la fecha

  4. Usando date con formato date: MMMM

    {{customDate | date:'MMMM'}}

  5. Usando date con formato date: MMMM dd, yyyy

    {{customDate | date:'MMMM dd, yyyy'}}

    6. Podemos cambiar el idioma de las fechas

  6. Usando date con formato date: long : '' : 'es-MX'

    {{ customDate | date:'long':'':'es-MX' }}
